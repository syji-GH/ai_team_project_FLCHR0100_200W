*   **後端框架**：使用 FastAPI (Python 3.12) 提供高效、非同步的 API 服務。

## 🛠️ 如何啟動本專案？

### 🔹 後端 (Back-end)
```powershell
cd dms/dms-backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 🔹 前端 (Front-end)
```powershell
cd dms/dms-frontend
npm run dev
```
